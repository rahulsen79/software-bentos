# Notion AI

Notion has three reusable things, and they do different jobs.

## Where these files go

- **instructions/**: paste into a skill or agent, or keep as a reference-library page that
  skills and agents point at. Notion does not read this repo directly, so treat the repo
  as the source of truth and keep the Notion pages in sync with it.
- **skills/**: a Notion skill is a page of reusable instructions. Create the page, open its
  ··· menu, choose **Use with AI**, then **Use as AI skill**. Invoke it by @mentioning it
  in chat. The markdown files here are the source you paste in. Manage skills under
  **Settings, Notion AI, General**.
- **agents/**: Custom Agents. One agent, one job. Recreate or audit a live agent from the
  definition file. Note that Custom Agents are Business/Enterprise only and run on credits.
