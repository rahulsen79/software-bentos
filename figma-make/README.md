# Figma Make

## Where these files go

- **instructions/**: upload into Make's Guidelines folder. At the top of Make click
  **Code**, open the **guidelines** folder in the left file explorer, then **Upload** or
  drag the `.md` files in. Make applies them now and in future iterations. You can upload
  several, but check them for conflicts first; Make has no conflict resolution.
- **skills/**: `SKILL.md` files, one reusable task each.

## Good for / not for

Good for fast 0 to 1 generation and ideation. Not a primary tool when the work needs to
round-trip cleanly with design-system-linked Figma files. Treat output as a draft.
